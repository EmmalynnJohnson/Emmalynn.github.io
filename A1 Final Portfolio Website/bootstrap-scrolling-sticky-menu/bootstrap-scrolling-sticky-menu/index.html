AI Project

In this project, I learned how to write effective prompts for Chat GPT. I worked with a group to create a portfolio for an author of children's books. An important skill I gained was learning to adjust the prompts I gave to Chat GPT to get the best results possible.

British Literature Tour Brochure

I collaborated with a group to create a brochure for a British Literature Tour run by Brigham Young University-Idaho. The purpsoe of the brochure is to provide information and to persuade students to come on the tour. The skill I learned for this project was Adobe Indesign.

Poster Design

This project honed my skill with photo shop. The first poster I made to advertise for a ballroom dance class that I teach to local high schoolers. The second poster, advertising a non-existant hippie party, I made for fun to experiment with different elements of photo shop. I learned to use layers, selecting the dancers and masking the background so only the dancers showed up. I also used AI generative fill to add a dark cloud at the bottom of the dancers so that it didn't look too choppy. For the hippie party poster, I learned to blend layers after masking them. I also used generative AI to make the creepy moon face. I had a lot of fun playing with the text features, as well.
